## Installing / Getting started

A quick introduction of the minimal setup you need to running.

```shell
yarn
```

To install the dependecies.

```shell
yarn android
```

To run in android emulator or device.

```shell
yarn ios
```

To run in ios emulator.

```shell
yarn test
```

To run tests.
